# Ali Faizan — Signal / Noise SOC Portfolio

A premium SOC Analyst L1 portfolio built around the concept **Signal / Noise**.

## Concept

The portfolio tells a simple cybersecurity story:

```text
Noise → Triage → Evidence → Verdict → Hire
```

## Features

- Strong recruiter-first hero with clear SOC positioning.
- Integrated black-and-white portrait asset: `assets/ali-faizan-portrait.webp`.
- Proof console with TryHackMe, GitHub, LinkedIn, and learning stats.
- Three SOC case studies written like analyst notes.
- GSAP ScrollTrigger background transitions and reveal timing.
- Purposeful parallax depth: foreground, evidence visuals, and background signal fields.
- Recursive variable heading morphing on scroll.
- Cursor-driven light, hidden scrollbar, and top progress indicator.
- Lenis smooth scroll.

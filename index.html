Failed login and brute-force detection workflow.

Problem

Solution

Review source IP, affected user, timestamps, expected baseline, reputation, and event pattern before deciding severity.

Project 03 · Windows Logs

Suspicious activity timeline with MITRE mapping.

Problem

Single Windows events are often weak alone. The real signal appears when user, host, process, and network activity are connected.

Solution

Review Windows/Sysmon-style events, order them by time, identify suspicious process behavior, and map to MITRE tactics.
